Course Description

A practical introduction to databases and Web app development.

  1. Databases: terminology and applications
  2. creating, querying and updating databases
  3. the entity-relationship model for database design
  4. Web documents and applications: static and interactive documents
  5. Web servers and dynamic server-generated content
  6. Web application development and interface with databases

Prerequisite Some experience with programming in an imperative language such as Python, Java or C.

Exclusion This course may not be taken after - or concurrently with - any C- or D-level CSC course.
